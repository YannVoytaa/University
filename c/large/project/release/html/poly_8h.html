Dokumentacja definicji typów

◆ Mono

typedef struct Mono Mono

To jest struktura przechowująca jednomian.

Jednomian ma postać \(px_i^n\). Współczynnik \(p\) może też być wielomianem nad kolejną zmienną \(x_{i+1}\).

◆ Poly

typedef struct Poly Poly

To jest struktura przechowująca wielomian.

Wielomian jest albo liczbą całkowitą, czyli wielomianem stałym (wtedy arr == NULL), albo niepustą listą jednomianów (wtedy arr != NULL).

◆ PolyAddWithCoeff()

Poly PolyAddWithCoeff ( const Poly *  p,
const Poly *  q,
poly_coeff_t  c 
)

Dodaje pierwszy wielomian do drugiego, pomnożonego przez współczynnik.

Parametry
[in]p: wielomian \(p\)
[in]q: wielomian \(q\)
[in]c: współczynnik c
Zwraca
\(p\) + c * \(q\)

jeśli q jest współczynnikiem, a p nie, to chcemy stworzyć wielomian p + (q * c) = (q * c) + p * 1

ponizej zakladamy, ze wielomiany nie sa wspolczynnikami i, że exp sa posortowane

jesli dodalismy juz wszystkie jednomiany z p lub z q

jesli istnieją jednomiany zarówno z p i z q

jesli współczynniki są takie same, to je dodajemy do siebie

◆ PolyAt()

Poly PolyAt ( const Poly *  p,
poly_coeff_t  x 
)

Wylicza wartość wielomianu w punkcie x.

Wstawia pod pierwszą zmienną wielomianu wartość x. W wyniku może powstać wielomian, jeśli współczynniki są wielomianami. Wtedy zmniejszane są o jeden indeksy zmiennych w takim wielomianie. Formalnie dla wielomianu \(p(x_0, x_1, x_2, \ldots)\) wynikiem jest wielomian \(p(x, x_0, x_1, \ldots)\).

Parametry
[in]p: wielomian \(p\)
[in]x: wartość argumentu \(x\)
Zwraca
\(p(x, x_0, x_1, \ldots)\)

◆ PolyDegBy()

poly_exp_t PolyDegBy ( const Poly *  p,
size_t  var_idx 
)

Zwraca stopień wielomianu ze względu na zadaną zmienną (-1 dla wielomianu tożsamościowo równego zeru).

Zmienne indeksowane są od 0. Zmienna o indeksie 0 oznacza zmienną główną tego wielomianu. Większe indeksy oznaczają zmienne wielomianów znajdujących się we współczynnikach.

Parametry
[in]p: wielomian
[in]var_idx: indeks zmiennej
Zwraca
stopień wielomianu p z względu na zmienną o indeksie var_idx
